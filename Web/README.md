#### Wordpress

* Wordpress Ubuntu single site :heavy_check_mark:

```
wget  -cO - https://raw.githubusercontent.com/reisikei/docker/main/Web/Wordpress > wp.sh && chmod 775 wp.sh && sudo ./wp.sh
```
